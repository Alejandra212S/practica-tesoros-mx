# Tesoros Escondidos de México

Página web informativa sobre lugares naturales, históricos y culturales de México.

## Archivos incluidos

- `login.html`: pantalla de inicio de sesión.
- `registro.html`: pantalla de registro.
- `index.html`: página principal con los destinos.
- `style.css`: estilos generales del sitio.
- `auth.js`: funcionamiento del registro, login y cierre de sesión con localStorage.
- `bacalar.html`: página de Bacalar.
- `tolantongo.html`: página de Grutas de Tolantongo.
- `xilitla.html`: página de Xilitla.
- `real-catorce.html`: página de Real de Catorce.
- `hierve-agua.html`: página de Hierve el Agua.
- `las-coloradas.html`: página de Las Coloradas.
- `img/`: carpeta con las imágenes usadas en el sitio.

## Cómo probarlo

1. Abre `registro.html`.
2. Crea una cuenta.
3. Inicia sesión en `login.html`.
4. Entra a la página principal y navega por los destinos.

## Nota

El inicio de sesión funciona con `localStorage`, por lo que es una simulación para proyecto escolar. No debe usarse como sistema de seguridad real.
